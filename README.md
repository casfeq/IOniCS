# IOniCS

IOniCS intends to provide tools for FEniCS and multiphenics problem solving environments.

## Dependencies & Installation

- [Python 3](https://www.python.org/downloads/) (3.8.2);
- [FEniCS](https://fenicsproject.org/download/) (1:2019.1.0.3);
- [matplotlib](https://pypi.org/project/matplotlib/) (3.1.2);
- [multiphenics](https://github.com/mathLab/multiphenics);
- [numpy](https://pypi.org/project/numpy/) (1.17.4).

You may install IOniCS by running the command:

```bash
./install.sh
```

## Uninstallation

You may remove this package by running:

```bash
./uninstall.sh
```
